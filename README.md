# acessosweb
Software para visualizar os logs dos acessos à Internet dos usuários e apresentar os dados em um Dashboar moderno. Os logs são armazenados pelo Squid em uma base de dados Mysql.
Projeto desenvolvido com as tecnologias: Java Web, Apache Tomcat/8.5.6, Mysql e Squid Cache.
